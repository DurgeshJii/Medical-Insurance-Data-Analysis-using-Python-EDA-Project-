# Medical-Insurance-Data-Analysis-using-Python-EDA-Project-
This project focuses on exploratory data analysis (EDA) of an insurance dataset to uncover insights related to medical insurance charges and the factors affecting them. The analysis demonstrates the complete data analytics workflow, including data cleaning, feature understanding, visualization, and insight generation using Python.
The dataset contains demographic and lifestyle attributes such as age, gender, BMI, smoking status, number of children, and region, along with corresponding insurance charges.
